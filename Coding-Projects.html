Why Coding?

Coding is important for learning to innovate, create eco-friendly solutions for global problems, important in our daily life to enhance and increase the power of computers and internet. Also important to automate, collect, manage, calculate, analyze processing of data and information accurately.

Coding is a process of 3 important phases

  1. Problem Identification and solving Phase
  2. Implementation Phase
  3. Maintenance Phase

All of them are key for software developer to solve a problem.

Reasons to learn Coding

  • Coding improves problem solving skills by teaching you how to think
  • Coding develops logical thinking by encouraging one to use logic and algorithm to create a program
  • Coding fosters creativity
  • Coding helps one understand how software works
  • With rise in technology there is a demand for software engineers, programmers and also developers. With coding as a fundamental skill you can open up your career choice
